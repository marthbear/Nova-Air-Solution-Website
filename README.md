Nova Air Solution
A business website for Nova Air Solution, built with Next.js, React, TypeScript, and Tailwind CSS.

Pages
Home — Main landing page with company overview
About — Information about the company
Services — Service offerings
Service Area — Geographic coverage area
Contact — Contact form and information
Service Requests — Customer service request submissions
Tech Stack
Next.js 16
React 19
TypeScript
Tailwind CSS 4
Firebase
Getting Started

npm install
npm run dev
Open http://localhost:3000 to view the site.

Scripts
Command	Description
npm run dev	Start development server
npm run build	Build for production
npm run start	Start production server
npm run lint	Run ESLint
